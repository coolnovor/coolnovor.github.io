Hexo常用命令

💡 Tips：本地调试渲染并预览无误后再进行Gitee推送

  • 新建草稿 hexo new draft ==** hexo n draft 草稿名称**
  • 草稿生成文章 hexo publish ==** hexo p 名称**
  • 新建文章 hexo new ==** hexo n 文章名称**
  • 清除已生成的预览缓存 hexo clean ==** hexo c ↓**
  • 生成页面静态文件 hexo generate ==** hexo g ↓**
  • 运行本地服务器 hexo server ==** hexo s ↓**
  • 部署到服务器 hexo deploy ==** hexo d ↓**
